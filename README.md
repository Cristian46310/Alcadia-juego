# trafic-rider-alcadia
Juego simuladando trafic rider, con la ciudad de manizales, para trabajo de la alcadia

## Inicio rapido

El repositorio esta preparado para guardar solo lo esencial del proyecto de Unity. Al abrir `My project/` en Unity, el editor vuelve a crear automaticamente `Library/`, `Temp/`, `Logs/` y el resto de archivos generados.

### Dependencias

Las dependencias del proyecto se cargan desde `My project/Packages/manifest.json`. No hace falta versionar las carpetas regenerables; basta con abrir el proyecto en Unity y dejar que instale lo que pida al iniciar.
